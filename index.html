Three main differences between Web1, Web2 and Web3

Web 1.0 is a read-only version. The websites contain information for searching but the interaction with the user is very limited. Web 2.0 is meant for data analyst. In this version the websites can have an interaction with the costumers. Users can share and write information (for instance: Wikipedia), and not only read it. Web 3.0 is the "semantic internet", which means: now the internet can analyze the meaning of the users' search words. This is actually an upgradation of the back-end, after version that mainly focused on the front.
